## **php CRUD**

使用 php PDO 來製作 CRUD

- 製作了[會員]()、[商品]()及[討論區]()的管理列表

### 新增
- 點擊新增會員、新增商品、新增文章的按鈕，輸入各欄位的相關資訊，點擊新增按鈕後，網頁會回到管理列表，新增的欄位會出現在列表的最上方。
